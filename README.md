## Commands for development

- Install dependencies:
  ```
  npm i
  ```
- Launch the app:
  ```
  npm start
  ```
- Create a new element in src folder:
  ```
  npm run create:element
  ```
- Run the tests with coverage output:
  ```
  npm test
  ```
